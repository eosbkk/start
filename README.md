## Welcome to EOS BKK
EOS BKk is a project and meetup for anyone interested in learning about the EOS blockchain.

การพบปะกับ EOS Bangkok ทำให้คุ้นเคยกับ EOS และเรียนรู้เกี่ยวกับเทคโนโลยี blockchain
